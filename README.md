# PDF to Black & White Converter

Convert colored PDFs to pure black and white for display e-ink device.

## Features

Upload PDFs with yellowed/tinted pages
Adjustable threshold for black/white conversion
Preview converted pages before download

## Usage

1. Select a PDF file
2. Adjust the threshold slider (default: 128)
3. Click "Convert to B&W"
4. Preview and download

Note: Lower threshold means more black pixels and higher threshold means more white pixels.
